Instalação:

1) Instale o Composer na máquina. Ver https://getcomposer.org/doc/00-intro.md
2) Entre dentro da pasta webiblio e digite: composer update
3) Crie uma banco de dados com acesso para criação de tabelas para um usuário. De preferência MySQL 
mas pode ser SQLLite ou SQL Server.
4) Configure o acesso ao banco de dados a partir do arquivo .env . No arquivo webiblio/config/database.php 
vc tem as configurações validas para diversos bancos.
5) Na pasta do webiblio execute o comando: php artisan migrate . Este comando criará as tabelas no banco escolhido.
7) Acesse http://[seulocal]/webiblio/public para usar o sistema.
